.. _cli_show:

show
====

Shows the FITS files.

------------

.. method:: im show

    **usage**

        im show [-h] file

    **positional arguments**

        ``file``        A file path or pattern (e.g., "\*.fits")


    **options**

        ``-h``, ``--help``  show this help message and exit

------------

Example:
________

.. code-block:: bash

    im imstat /PATH/TO/FILES*.fits
