MYRaf documentation
===================

The MYRaf WAS a practicable astronomical image reduction and photometry software and interfaces for IRAF. Nowadays MYRaf uses Astropy and other pythonâ€™s libraries to achieve its purpose. For this, MYRaf uses Astropy, Astroquery, and many other python packages with a Qt framework. Also, MYRaf is free software and distributed with a GPLv3 license.

Installation
------------

The installation of MYRaf can be accomplished through the Python package manager by executing the following command:

.. code-block:: bash

    pip install myraf

For further information, including source code, please visit the project's homepage.
