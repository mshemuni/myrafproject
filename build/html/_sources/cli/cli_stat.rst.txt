.. _cli_stat:

stat
====

Return image statistics of FITS files.

------------

.. method:: im stat

    **usage**

        im stat [-h] file

    **positional arguments**

        ``file``        A file path or pattern (e.g., "\*.fits")


    **options**

        ``-h``, ``--help``  show this help message and exit

------------

Example:
________

.. code-block:: bash

    im imstat /PATH/TO/FILES*.fits
